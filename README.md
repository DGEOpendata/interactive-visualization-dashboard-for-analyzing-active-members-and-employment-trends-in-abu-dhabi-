# Interactive Visualization Dashboard for Analyzing Active Members and Employment Trends in Abu Dhabi for 2022

## Overview
This project provides an interactive visualization dashboard to explore and analyze the 'Active Members Trends in 2022' dataset. The dashboard allows users to filter and visualize data related to active members, including their demographic distributions, sectoral employment, and years of service. The dashboard is built using Python, Flask, and Plotly.

## Features
- Visualize trends in employment and demographics using interactive charts.
- Filter data by quarter, age group, gender, sector, and years of service.
- Download visualizations in various formats for offline analysis.
- User-friendly interface suitable for diverse audiences, including policymakers, researchers, and businesses.

## Requirements
- Python 3.8 or higher
- Flask
- Pandas
- Plotly

Install dependencies using pip:
bash
pip install flask pandas plotly


## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/abu-dhabi-labor-dashboard.git
   cd abu-dhabi-labor-dashboard
   

2. Place the dataset file (`Distribution_of_Active_Members_2022_0.xlsx`) in the project directory.

3. Run the Flask app:
   bash
   python app.py
   

4. Open your web browser and navigate to `http://127.0.0.1:5000/` to access the dashboard.

## Usage
1. Open the dashboard in your preferred web browser.
2. Use the dropdown menus and filters to select the desired data subsets (e.g., by quarter, age group, gender, etc.).
3. Interact with the visualizations to explore trends and insights.
4. Download the visualizations as images for use in reports or presentations.
